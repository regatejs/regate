Basic #

This is the most basic usage of RegateNumber. 

Regate.RegateNumber.init({
  id: 'basic-example',
  name: 'Total',
})

Value #

Regate.RegateNumber.init({
  id: 'value-example',
  name: 'Total',
  value: 5,
})

Required #

Regate.RegateNumber.init({
  id: 'required-example',
  name: 'Total',
  required: true,
})

Min #

Regate.RegateNumber.init({
  id: 'min-example',
  name: 'Total',
  min: 10,
})

Max #

Regate.RegateNumber.init({
  id: 'max-example',
  name: 'Total',
  max: 10,
})

Step #

Regate.RegateNumber.init({
  id: 'step-example',
  name: 'Total',
  step: 0.5,
})

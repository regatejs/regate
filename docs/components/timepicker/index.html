TimePicker
  • v0.16.0
This component has dependencies. Read the below section for more information.

Dependency

This component depends on flatpickr. Be sure to include it on your page.

<script src="https://cdn.jsdelivr.net/npm/flatpickr@4.5.2/dist/flatpickr.min.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/flatpickr@4.5.2/dist/flatpickr.min.css" />

Basic #

This is the most basic usage of RegateTimePicker. 

Regate.RegateTimePicker.init({
  id: 'basic-example',
  name: 'Hour',
})
@Html.Raw(RegateTimePicker.Build("Hour"))

Value #

Regate.RegateTimePicker.init({
  id: 'value-example',
  name: 'Hour',
  value: '17:15:00'
})
@Html.Raw(RegateTimePicker.Build("Hour", new TimeSpan(17, 15, 00)))

Required #

Regate.RegateTimePicker.init({
  id: 'required-example',
  name: 'Hour',
  isRequired: true,
})
@Html.Raw(RegateTimePicker.Build("Hour", isRequired: true))
